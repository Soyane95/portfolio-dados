# 📊 Projeto de Análise de Vendas

## 📌 Objetivo
Analisar dados de vendas para gerar insights sobre desempenho de produtos, regiões e faturamento.

## 🔧 Ferramentas
- Google Sheets

## 📊 Análises realizadas
- Faturamento por produto
- Vendas por período
- Distribuição por região

## 📈 Resultados
Identificação de produtos com maior desempenho e regiões com maior volume de vendas.
